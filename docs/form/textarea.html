프로트엔드 개발을 위한 HTML 학습하기

textarea

내용물을 가지는 입력양식 입니다.

textarea는 시작테크와 종료 테그로 구성이 됩니다.

<textarea name="" id="" cols="30" rows="10">안녕하세요</textarea>

크기설정

넓이 cols

사용 폰트나 언어에 따라서 입력되는 넓이의 크기는 다를 수 있습니다.

줄수 rows

내용물

기본값을 입력할 경우, html 작성코드와 들여쓰기 내용도 같이 들어가게 됩니다.

textarea

여러줄의 글씨를 입력할 수 있는 요소 입니다. 다중 편집창

입력창의 크기를 지정할 때는

  • cols : 가로 텍스트 입력수, 가로폭

  • rows : 세로 입력 수 , 높이 엔터의 갯수를 의미합니다.
