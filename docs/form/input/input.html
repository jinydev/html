type 형식 속성

input은 다양한 속성값을 부여하여 처리를 할 수 있습니다.

text

<input type="text">

submit

입력롬을 서버로 전송을 위한 버튼을 출력합니다.

<input type="submit">

checkbox

라벨과 연동하여 체크를 처리할 수 있습니다.

file

hidden

image

password

값을 입력할때 * 로 대체되어 표시가 됩니다.

radio

여러개의 중에서 하나만 선택할 수 있습니다. 주의할 점은 name 속성을 동일하게 부여를 하여야 합니다.
