# 테이블 테그

## 학습목표
---
* table 태그 구조에 대해 살펴볼 수 있다.
* table 태그의 속성들을 살펴볼 수 있다.
* th,tr,td 태그에 대해 살펴볼 수 있다.
* 테이블의 셀 병합하는 방법을 익힐 수 있다.

## 기본 테그

### 테이블 태그 기본
표를 만들 때는 table 태그를 시용 합니다. 
* table
* tr : 표 내부의 행 태그
* td : 행 내부의 제목 셀 태그
* th : 행 내부의 일반 셀 태그

### 표 영역 요소
* thead
* tbody
* tfoot
* caption

### 행, 열 합치기
* colspan
* rowspan
* col
* colgroup

## 테이블 속성
예전에는 table 태그는속성이 굉장히 많았습니다. 하지만, HTML5에서는 속성에서 처리하는 설정값을 CSS3로 대체가 되고 있습니다.

* border
* rowspan : 셀의 높이 지정
* colspan : 셀의 너비 지정
