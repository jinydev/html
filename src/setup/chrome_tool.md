# 개발자 도구
최신 인기있는 브라우저들은 자체적으로 HTML/CSS 를 분석할 수 있는 개발자 도구를 같이 제공을 합니다.

## 크롬 설치
웹 개발자들이 크롬 브라우저를 선호하는 이유는 최신의 HTML, CSS 표준을 따르는 것과 편리한 개발자도구를 같이 제공하고 있기 때문입니다.

* 구글 크롬을 설치합니다.



## 개발자 도구 실행
크롬 브라우저에서 `F12` 키를 누르게 되면 오른쪽에 크롬 개발자 도구가 같이 표시가 됩니다.
